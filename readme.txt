Мы написали приложение, представляющее собой чат-бота на языке программирования Python, содержащий информацию о расписании занятии в ЦЦОД IT-куб. 
Основная особенность нашего бота заключается в том что в нём представлена функция Админ-Панели.
С помощью неё Админ (человек которому доверили управление Админ-Панелью) может изменять (добавлять/убирать) дни учёбы/время/кабинет/преподователя.

Для запуска бота вы должны скачать файлы из GitHub и переместить их в одну папку,
запускаете файл через PyCharm или cmd с помощью команды 'python main.py' в директории в которой находится скачанные вами файлы
